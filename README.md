# Decova

> Decode what you see. Discover how it works. Design with it.

Decova is a browser extension built for designers and developers who are constantly collecting inspiration but missing the *context behind it*.

Instead of saving static screenshots or digging through complex DOM trees, Decova lets you **capture UI elements and instantly decode their structure, styles, and intent**—turning visual inspiration into something you can actually use.

This is not just about saving what looks good.
It’s about understanding *why* it works.

---

### ✨ What Decova Does

Decova bridges the gap between **inspiration and implementation**.

* **Decode**
  Extract meaningful design details from any UI element without manually inspecting layers.

* **Discover**
  Build a personal library of patterns, components, and styles you can revisit and learn from.

* **Capture with Context**
  Every saved element includes both the visual reference and the structured data behind it.

---

### 🧩 Core Features

#### 🎯 Smart Capture

Select any element on a live website with a simple hover + click interaction.
Decova intelligently identifies the most relevant component, so you don’t have to fight the DOM.

#### 🎨 Design Intelligence

Automatically extracts:
* Typography (font family, size, weight)
* Colors (text, background)
* Spacing and layout
* Border radius, shadows, and visual effects

All organized in a way that actually makes sense to humans.
